# MotorPH Payroll System - Milestone 2

## Project Description
A Java-based payroll application that automates salary calculations for MotorPH employees. The system integrates employee data and attendance records from CSV files to generate semi-monthly payroll reports.

## Features
- **Secure Login:** Access restricted to authorized staff.
- **CSV Integration:** Dynamic data retrieval from `Employee Details` and `Attendance Record`.
- **Accurate Calculations:** Automated SSS, PhilHealth, Pag-IBIG, and Withholding Tax based on official 2024 tax brackets.
- **Report Generation:** Full payroll breakdown for June through December.

## How to Run
1. Run `Main.java` in your IDE.
2. Login with:
   - **Username:** `employee`
   - **Password:** `12345`
3. Enter an Employee ID (e.g., `10001`) to view the report.
4. Type `exit` to close the program.
